You’ll need:
   - Bolt IOT wifi module
   - Temperature sensor
   - Jumper wires
   - Bolt IOT mobile app

*************************************************************/

1 Login into https://cloud.boltiot.com and click on the 'Product' tab.
2 Create a new product for your temperature monitoring system.
3 Click on Configure this product to configure the product.
4 Click on the "A0" pin of the Bolt and give it a name in the right side naming section.
5 Click on the Code Section then click on the "Import Code example" icon.

code will be look like-

setChartLibrary('google-chart');

setChartTitle('Your Graph Title');

setChartType('lineGraph');

add(183);
mul(0.0977);
mul(1.8);
plotChart('time_stamp','temperature');


6 now write the file name of the code and choose extension .js.

Finally, click on the "Save" icon to save the changes to the product.

After linking the Bolt device to the product. Deploy the code and click to the plot.
